-----环境
我运行程序用的是Matlab 2016b版本

-----目录结构
\data---放的是.mat格式的数据
\methods---放的是各个迁移学习方法的实现，其中包括的对应的paper
\output---运行demoXXX.m后得到的结果
\tools---一些工具类

------demo文件
demoXXX.m用于运行对应的算法，对应的手调参数请自行设置

目前版本包含的方法如下：（除了JDM要配置cvx之外其他可直接运行，不会配置的话可以联系我：qiushaojian@outlook.com）
1、DG
2、NNFilter
3、KMM
4、TCA
5、JDA
6、JDM--该方法的运行请参考./methods/JDM下的How_to_run_JDM.md
